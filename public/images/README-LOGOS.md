## Replacement Instructions

To replace with official WASL logos from AMAN Innovatia:

1. **Obtain official logos** from AMAN Innovatia in the following formats:
   - SVG (preferred for web)
   - PNG (high resolution, with transparent background)
   - ICO (for favicon)

2. **Replace placeholder files:**
   ```bash
   # Main logo
   cp official-wasl-logo.svg public/images/wasl-logo.svg
   cp official-wasl-logo.png public/images/wasl-logo.png

   # Dark mode variant (if provided)
   cp official-wasl-logo-dark.svg public/images/wasl-logo-dark.svg
   cp official-wasl-logo-dark.png public/images/wasl-logo-dark.png

   # Favicon
   cp official-wasl-favicon.ico public/favicon.ico
   ```

3. **Required Logo Variants:**
   - Standard logo (light backgrounds)
   - Dark mode logo (dark backgrounds) - optional
   - Favicon (16x16, 32x32, 48x48 ICO)
   - High-resolution PNG (minimum 512x512)
